# Life Cycle Assessment of Fresh Tea Leaf Production

This repository contains a Life Cycle Assessment (LCA) model of fresh tea leaf production developed using **openLCA**.

## Project Overview

The objective of this project is to evaluate the environmental impacts associated with producing **1 kg of fresh tea leaves** by modeling the complete production system.

## Functional Unit

- **1 kg of fresh tea leaves**

## System Boundary

The model includes the following processes:

- Tea cultivation
- Irrigation
- Diesel production
- Electricity generation
- Machinery use
- Water processing
- Pesticide production
- Urea production
- Phosphate fertilizer production
- Potassium fertilizer production

## Software Used

- openLCA 2.6.2

## Repository Contents

- JSON-LD export of the openLCA model
- Inventory data
- Calculation results
- Process screenshots
- Contribution analysis
